# Pneumonia Diagnosis System

Bu proje, göğüs röntgeni (Chest X-ray) görüntülerinden derin öğrenme modelleri kullanarak zatürre (pneumonia) tespiti yapmayı amaçlayan bir bilgisayarlı görü projesidir.

## Proje Hakkında
*   **Amaç:** Medikal görüntü analizi ile hızlı ve güvenilir bir ön tanı destek sistemi oluşturmak.
*   **Model:** ConvNeXtV2 Base mimarisi.
*   **Teknoloji:** Python, PyTorch, Streamlit.

## Özellikler
*   Yüksek doğruluk oranı ile sınıflandırma.
*   Kullanıcı dostu arayüz (Streamlit).
*   Görüntü ön işleme ve analiz modülleri.
